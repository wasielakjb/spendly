# Spendly

## Project Description
Spendly is a simple mobile application designed for adding and monitoring personal expenses. The application enables users to maintain continuous control over their household finances by recording expenses and analyzing them across selected time periods.

The project is designed as an application with a straightforward structure and an intuitive user interface, intended for individuals who are beginning to manage their household budget.


## Getting Started
1. Clone the repository.
2. Install dependencies using `flutter pub get`.
3. Run the application on an emulator or a physical device.
